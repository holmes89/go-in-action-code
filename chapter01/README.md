# Chapter 1 Examples

This directory contains the code examples from Chapter 1: Introducing Go.

## Examples

- **type_safety.go** - Demonstrates Go's type safety (contains commented code that won't compile)
- **goroutine_basic.go** - Basic goroutine syntax
- **goroutine_slow.go** - Running a slow function concurrently
- **channels.go** - Using channels to exchange data between goroutines
- **struct_basic.go** - Basic struct definition
- **struct_methods.go** - Struct with methods
- **interfaces.go** - Interface example with Person and Cat types
- **generics.go** - Generic programming with type parameters

## Running the Examples

You can run any example with:

```bash
go run <filename>.go
```

For example:
```bash
go run goroutine_slow.go
go run interfaces.go
go run generics.go
```
